## 🛠️ How to Run

### Prerequisites
- Node.js 18+
- npm or yarn
- Expo CLI
- Expo Go app (for mobile testing)

### Backend Setup
```bash
cd backend
npm install
npm run dev
```

### Frontend Setup
```bash
cd mobile
npm install
npm start
```

### Testing on Expo Go
1. Install Expo Go app on your phone
2. Scan the QR code from the terminal
3. The app will load and work immediately
